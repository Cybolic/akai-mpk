AKAI – MPK mini

Important note: For the script to work, you need to load the Bitwig Studio preset into the MPKmini using the Akai editor.
CC + PROG CHANGE OFF

Play pads to trigger notes.

CC ON
PAD BANK 1
P 5/1 Previous/next Track = Cursor Track.
P 6/2 Previous/next Device = Cursor Device.
P 7/3 Transpose Pads up/down.
P 8 Toggle Macro/Device Mapping Mode for Knobs K 1-8
P 4 In Device Mapping Mode: Cycle through User Device Maps
PAD BANK 2
P 5 Transport: Stop.
P 6 Transport: Play.
P 7 Transport: Record.
P 8 Toggle Overdub.
P 1 Arm the current Track.
P 2 Solo the current Track.
P 3 Mute the current Track.
P 4 Toggle Launcher Overdub.
PROG CHANGE ON
PAD BANK 1
P 5/1 Select previous/next Preset.
P 6/2 Select previous/next Preset Category.
P 7/3 Select previous/next Preset Creator.
P 8 Toggle Macro/Device Mapping Mode for Knobs K 1-8
P 4 In Device Mapping Mode: Cycle through User Device Maps
PAD BANK 2
P 5 Toggle Inspector.
P 6 Toggle Perspective
P 7 Cycle through Project Tabs
P 8 Select/Toggle Browser
P 1 Select/Toggle Note Editor Panel.
P 2 Select/Toggle Automation Editor Panel.
P 3 Select/Toggle Device Panel.
P 4 Select/Toggle Mixer Panel.
GENERAL
K 1-8 Macros or Device Mappings of the Cursor Device, depending on Mode settings.
I/O Panel Drum Pad Transpose and Knob Mode are saved per Project.
See the Bitwig Studio I/O Panel for the settings.
